## First Readme

```ECommerce Website```

## Description

This is a simple ECommerce website that allows users to view products, add them to cart and checkout. The website is
built using HTML, CSS and JavaScript. The website is responsive and can be viewed on mobile devices as well. The website
is hosted on GitHub Pages.

## Features

- View products
- Add products to cart
- Remove products from cart
    - Increase quantity of products
    - Decrease quantity of products
- Calculate total price of products in cart
- Checkout
